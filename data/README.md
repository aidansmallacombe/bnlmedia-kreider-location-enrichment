Per the licensing agreement from SimpleMaps, I cannot distribute the 'uszips' database publicly and have thus omitted it from the project using .gitignore.

The database is available for free by downloading it on https://simplemaps.com/data/us-zips. The code for this project expects the database to be extracted to 'data/reference', so a 'reference' folder would need to be created in this directory for proper setup.
